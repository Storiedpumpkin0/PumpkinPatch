Download files below


  Here various completed vagrant boxes can be downloaded, as well as LANL-specific vagrant files for those boxes.
  A vanilla Vagrantfile can be generated for each box by simply running the following command: 
    vagrant init http://link_to_box_download

  Boxes with the latest updates can be found at: http://rntest.lanl.gov/boxes/updated-boxes/
