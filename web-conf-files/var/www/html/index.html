About the project

DevOps cloud on the fly provides automated virtual machine creation on demand. This is accomplished using two very powerful software tools: Packer and Vagrant.

Packer by Hashicorp is a software which specializes in the creation, provisioning, and packaging of virtual environments from scratch. It does this by reading a .json file in which the developer can give specific options for the virtual environment. In this .json file there are options for building the VM, providing unattended install files, and provisioning with command line arguments and shell scripts. Additionally packer can package up the virtual machine into a number of forms after the process is completed including .iso files, virtual box images, vagrant boxes, and Docker files.

Vagrant, also by Hashicorp, is a software which can open, provision, and re-package virtual machines from pre-made boxes. These “vagrant boxes” can be compressed for easy portability and can run identically across multiple platforms. Vagrant works by reading a “Vagrantfile” in which the user can specify various build options including provisioning system specifications, environmental variables, configuring shared storage, running custom scripts, and executing command line arguments.


These two tools can be used in tandem to create any number of virtual machines on demand, and then destroy them completely when testing is finished.
